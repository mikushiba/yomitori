# よみとりたんてい — 小2 こくご 文しょうだい

小学2年生向けの国語文章題アプリ。子どもが「たんてい」になって、書き下ろしの短い文章を読み、3つの「なぞ」(設問)に答える。全24話(物語12話+説明文12話)×3問=72問。各章(だい1〜3しょう)に物語4話+説明文4話。

- 公開URL: https://mikushiba.github.io/yomitori/
- 本体: `index.html` 1ファイルで完結。サーバー不要、`file://` で直接開いても動く。

## 設計思想

1. **答えを勘で選ばせない。** 1回まちがえると、正解を教える代わりに本文中の「しょうこの文」に黄色い目じるしを付けて再挑戦させる。「答えは文章の中にある」という読解の基本動作を、仕組みそのもので教える。
2. **設問を6タイプに分類。** だれ・なに／じゅんばん／きもち／りゆう／こそあど／だいじなこと。初回正解できたタイプを記録し、「おうちの人へ」画面でタイプ別の正答率バーと、いちばん弱いタイプへの声かけアドバイスを表示する。
3. **教科書と同じ読み方に寄せる。** 全漢字ふりがな(2年生配当中心)、分かち書き、たてがき表示への切り替え、音読ボタン(1文ずつハイライトしながら読み上げ)。読み上げはブラウザ内蔵の音声合成なので通信不要。
4. **ごほうびは丸つけ文化で。** 初回正解で★、話を解き終えると朱色の「かいけつ」スタンプ、全問一発正解で花丸。記録はブラウザのlocalStorageに自動保存(消すボタンあり)。

## 話や問題の増やし方

`index.html` 内の `PASSAGES` 配列に、同じ形でひとかたまり追記するだけ(タイトル・本文の文の一覧・設問3つ・正解番号・しょうこの文の番号・解説)。
